# Tera Mera Bank Application

A JavaFX-based desktop banking application for managing accounts, deposits, withdrawals, fund transfers, and other banking services.

## Features
- Open New Account: Allows users to create new accounts with a unique ID, initial balance, and password protection.
- Deposit: Users can deposit funds into their accounts.
- Withdraw: Users can withdraw funds, provided they have sufficient balance.
- Fund Transfer: Supports secure fund transfer between two existing accounts.
- Check Balance: Users can check their account balance after providing the correct credentials.
- Close Account: Deletes an account from the system.

## Setup Instructions
1. Prerequisites: Ensure Java 8+ and JavaFX are installed.
2. Running the Application:
   - Clone the repository.
   - Compile and run the `ComboProj.java` file to start the application.

## Usage
Upon running the application:
1. Select a Service: Use the drop-down menu to select a banking operation.
2. Account Management:
   - For each service, follow the on-screen prompts to enter account details, transaction amounts, and other required information.
   - User passwords must be at least 8 characters long.

## Project Structure
- `Account.java`: Defines the `Account` class for managing account-related operations like deposits, withdrawals, and password verification.
- `ComboProj.java`: Main application file, implementing the JavaFX GUI and handling user interactions.

## Dependencies
- JavaFX: Used for the GUI.
- Java Standard Library: Utilizes collections, math functions, and basic Java utilities.
